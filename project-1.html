Matbestillings Applikasjon

En web app som gir brukeren mulighet til å legge inn nye oppskrifter og bestille retter. Bygget med C#, ASP.NET og Razor Pages.

Live Link

Om Prosjektet

Et skole prosjektet som er laget .NET 7.0. Applikasjonen er en mat bestillings applikasjon er lignende til Foodora og Wolt, der brukeren også kan legge til egne oppskrifter. Fokusert på at applikasjonen skal være generisk og testbar, og at classer og contexter ikke overlapper. Under er det vist til klassediagrammet til applikasjonen, linjene er api calls fra en pipeline til en service, striplete linjer er event baserte meldinger mellom contextene.

Applikasjonen bruker .NET og C# og Azure som web service. Se gjerne prosjektet i github eller live link.
